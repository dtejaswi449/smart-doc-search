# Smart Document Search

A Streamlit-based application that enables question answering over text-based PDF documents using semantic search and extractive QA.

## Features
- Upload and process text-based PDFs
- Chunking and vector indexing using FAISS
- Semantic retrieval with sentence-transformer embeddings
- Extractive question answering using HuggingFace Transformers
- Guardrails to detect scanned/image-only PDFs

## Tech Stack
- Python, Streamlit
- LangChain, FAISS
- HuggingFace Transformers

## Notes
This is an initial version focused on correctness and pipeline clarity. Accuracy improvements and source attribution are planned.
